# Anime management API - bash
## Install dependencies
- With yarn:
```bash
yarn install
```

- With npm:
```bash
npm install
```

## Run API
- With yarn:
```bash
yarn dev:docker
```

- With npm:
```bash
npm run dev:docker
```

# Anime management API - docker
- Turn the command on the terminal:
```bash
docker build -t backend-animerex .
```
